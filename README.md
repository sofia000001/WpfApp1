# Лабораторная работа №1 - Разработка пользовательского интерфейса (GUI) для языкового процессора

## Цель работы
Создание кроссплатформенного графического интерфейса (GUI) для языкового процессора в виде специализированного текстового редактора.

- **Группа:** АП-326


## Описание проекта
Текстовый редактор, разработанный в рамках лабораторной работы. 
Приложение является основой для будущего языкового процессора анализа исходного кода.

**Функциональность:**
- Создание, открытие, сохранение текстовых файлов
- Стандартные операции правки (отмена, вырезать, копировать, вставить)
- Интерфейс с изменяемыми областями
- Панель инструментов с иконками для быстрого доступа
- Область вывода результатов (подготовка для языкового процессора)


## Используемые технологии
- **Язык программирования:** C#
- **Фреймворк:** WPF (Windows Presentation Foundation)
- **Среда разработки:** Visual Studio 2022
- **Платформа:** .NET Framework 4.7.2

##  Руководство пользователя

### Интерфейс программы

![Главное окно программы](WpfApp1/Screenshots/MainWindow.png)

**Основные области:**
1. **Основное меню** - доступ ко всем функциям программы
2. **Панель инструментов** - быстрый доступ к часто используемым функциям (с иконками)
3. **Область ввода/редактирования** - текстовый редактор (поддерживает ввод, табуляцию, перенос строк)
4. **Область вывода** - отображение результатов (только для чтения)

### Функционал

#### Меню "Файл"
| Функция | Горячие клавиши | Кнопка на панели | Описание |
|---------|-----------------|-------------------|----------|
| Создать | Ctrl+N | Создать | Создаёт новый файл |
| Открыть | Ctrl+O | Открыть | Открывает существующий файл |
| Сохранить | Ctrl+S | Сохранить | Сохраняет текущий файл |
| Сохранить как | - | Сохранить как | Сохраняет файл с новым именем |
| Выход | Alt+F4 | - | Выход из программы (с подтверждением) |

#### Меню "Правка"
| Функция | Горячие клавиши | Кнопка на панели | Описание |
|---------|-----------------|-------------------|----------|
| Отменить | Ctrl+Z | Отменить | Отменяет последнее действие |
| Вырезать | Ctrl+X | Вырезать | Вырезает выделенный текст |
| Копировать | Ctrl+C | Копировать | Копирует выделенный текст |
| Вставить | Ctrl+V | Вставить | Вставляет текст из буфера |
| Удалить | Del | Удалить | Удаляет выделенный текст |
| Выделить всё | Ctrl+A | Выделить всё | Выделяет весь текст |

#### Меню "Справка"
| Функция | Горячие клавиши | Кнопка на панели | Описание |
|---------|-----------------|-------------------|----------|
| Справка | F1 | Справка | Показывает описание всех функций |
| О программе | - | О программе | Информация о программе |
