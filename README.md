# Текстовый редактор - Лабораторная работа №1

## Цель работы
Создание кроссплатформенного графического интерфейса (GUI) для языкового процессора в виде специализированного текстового редактора.

## Автор
- **Студент:** Дарчук Софья
- **Группа:** АП-326


## Описание проекта
Текстовый редактор, разработанный в рамках лабораторной работы. 
Приложение является основой для будущего языкового процессора анализа исходного кода.

**Функциональность:**
- Создание, открытие, сохранение текстовых файлов
- Стандартные операции правки (отмена, вырезать, копировать, вставить)
- Интерфейс с изменяемыми областями
- Панель инструментов с иконками для быстрого доступа
- Область вывода результатов (подготовка для языкового процессора)


## Используемые технологии
- **Язык программирования:** C#
- **Фреймворк:** WPF (Windows Presentation Foundation)
- **Среда разработки:** Visual Studio 2022
- **Платформа:** .NET Framework 4.7.2
